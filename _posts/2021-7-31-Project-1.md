# **Indexing**

## ***.loc vs .iloc***

### **.iloc return DATA at position number 'acording to POSITION'**

### **.loc return DATA at specific index number 'acording to INDEX number'**

    5     Panda
    6     Snake
    8    Turtle
    9       Cat
    2      Bird
    5    Rabbit
    dtype: object
    ==============
    Rabbit
    ==============
    5     Panda
    5    Rabbit
    dtype: object
    




    Make                Nissan
    Colour               White
    Odometer (KM)       213095
    Doors                    4
    Price            $3,500.00
    Name: 4, dtype: object
